# Base Portfolio Tracker

A simple Mini App for Farcaster that tracks your crypto portfolio on Base chain.

## Features

- 💼 View ETH, USDC, and WETH balances
- 🔗 Connect wallet via Farcaster Mini App SDK
- 🔄 Real-time balance refresh
- ⚡ Powered by Base L2

## Tech Stack

- Next.js 14
- Farcaster Mini App SDK
- Viem (Ethereum library)
- Base Chain

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/base-portfolio-tracker)

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Usage in Farcaster

1. Deploy to Vercel
2. Add manifest.json to your domain
3. Share the link in Farcaster
4. Users can open it as a Mini App
